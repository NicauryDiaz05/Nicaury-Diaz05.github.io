# Nicaury-Diaz05.github.io
Portafolio Digital
Informe de Elaboración del Portafolio Web Personal 
Este portafolio web fue desarrollado con el objetivo principal de presentar mi 
perfil profesional como estudiante de Ingeniería en Sistemas, mostrando mis 
habilidades técnicas, proyectos realizados y aspiraciones futuras en el campo 
tecnológico. Funciona como una plataforma digital que: 
Presenta mi identidad profesional de manera estructurada y atractiva 
Muestra mis capacidades técnicas en desarrollo web mediante la 
implementación del propio sitio 
Exhibe proyectos concretos que evidencian mis conocimientos aplicados 
Establece mi presencia digital como profesional en formación 
Facilita el contacto con potenciales empleadores o colaboradores 

Desarrollo Técnico

HTML5 semántico: Utilicé etiquetas semánticas para mejorar accesibilidad y 
SEO 
CSS3 avanzado: Implementé variables CSS, Grid, Flexbox y animaciones 
para crear una experiencia visual atractiva 
JavaScript funcional: Desarrollé interacciones como el menú móvil, modales y 
efectos de scroll suave 
Optimización de rendimiento: Código minificado, imágenes optimizadas y 
carga eficiente 


Sistema de diseño modular con variables CSS para consistencia visual 
El desarrollo de este portafolio web representa un avance importante en mi 
formación como Ingeniera en Sistemas, ya que no solo cumple con el objetivo 
de presentar mis habilidades y proyectos, sino que también demuestra mi 
capacidad para planificar, diseñar e implementar soluciones web completas y 
profesionales. Este proyecto evidencia mi compromiso con el aprendizaje 
continuo y mi potencial para contribuir en el campo del desarrollo de 
software.
